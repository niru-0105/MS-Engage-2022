# MS-Engage-2022
KillCrime is a web app that I have created in MS Engage 2022 mentorship program. It uses machine learning to recognise face present in a image and tells details about the criminal.
# Please follow steps written below to use this web app
1. If you do not have python installed on device please download it from here 'https://www.python.org/ftp/python/3.10.4/python-3.10.4-amd64.exe'. If you think you have python on your device please type this in command in terminal 'python -V'. If you see python version then it means you have oython installed on your device.
2. Open any terminal in your devise and type 'pip install django' to install django. If you think you already have then type 'python' now you have entred in python shell, type 'import django' then 'django.VERSION', if you do not get error then it means that django is already installed
3. Type 'pip install opencv-python' in terminal to install opencv.
4. Type 'pip install face_recognition' in terminal to install face_recognition.
5. Move to do directory where you have downloaded the code. Move to 'MS Engage 2022/Killcrime'. You should be in same directory where 'manage.py' is.
6. After moving to the correct directory type 'python manage.py runserver' in the terminal.
7. Now open browser and enter web address as 'localhost:8000' or 'http://127.0.0.1:8000/', now use the Killcrime web app and enjoy.
